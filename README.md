# ChurnLens 

ChurnLens is a modern **customer churn analytics dashboard** built with **Next.js, Tailwind CSS, and JavaScript**.
It helps businesses visualize customer retention trends, identify churn risks, and make data-driven decisions through an intuitive dashboard interface.

---

##  Features

*  Interactive churn analytics dashboard
*  Customer retention & loss trend visualization
*  Risk segmentation for churn prediction
*  Detailed customer insights
*  Fast and optimized Next.js frontend
*  Responsive UI with Tailwind CSS
*  Reusable modular components
*  Fully mobile-friendly design

---

##  Tech Stack

* **Framework:** Next.js
* **Language:** JavaScript
* **Styling:** Tailwind CSS
* **State / Utilities:** Custom utility functions in `lib`
* **Architecture:** Component-based scalable frontend

---

##  Project Structure

```bash
churnlens/
│── components/      # Reusable UI components
│── lib/             # Helper functions and utilities
│── pages/           # Next.js routes and pages
│── styles/          # Global and component styles
│── public/          # Static assets
│── package.json
│── tailwind.config.js
│── next.config.js
```

##  Installation

Clone the repository:

```bash
git clone https://github.com/Loveyyyy99/churnlens.git
cd churnlens
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:3000
```
